# 통합 로그라이크 게임

세 가지 독립적인 게임 시스템을 하나의 완전한 로그라이크 게임으로 통합한 프로젝트입니다.

## 🎮 게임 개요

**통합 로그라이크 게임**은 맵 탐험, 던전 크롤링, 턴제 전투가 하나로 통합된 웹 기반 게임입니다.

### 주요 특징

- **맵 시스템**: 노드 기반 경로 선택 (Slay the Spire 스타일)
- **던전 시스템**: Phaser 엔진 기반 실시간 던전 탐험 (WASD 이동)
- **전투 시스템**: 턴제 카드 전투 (공격/방어 선택)
- **자원 관리**: 금, 정보, 전리품, 원자재, 에테르
- **캐릭터 성장**: 체력, 에너지, 속도, 힘 스탯

## 📁 파일 구조

```
/home/user/ex/
├── game.html          # 🎯 통합 게임 메인 파일 (플레이용)
├── Dungeon.html       # 던전 시스템 원본
├── battle.html        # 전투 시스템 원본
├── map.html           # 맵 시스템 원본
├── 통합규칙.md        # 통합 작업 규칙
└── README.md          # 이 파일
```

## 🚀 시작하기

### 실행 방법

1. **웹 브라우저에서 `game.html` 파일을 엽니다**
   ```bash
   # 파일 탐색기에서 game.html 더블클릭
   # 또는 브라우저에서 직접 열기
   ```

2. **게임 플레이**
   - 맵 화면에서 시작
   - 파란색으로 빛나는 노드를 클릭하여 진행
   - 각 노드 타입에 따라 다른 화면으로 전환

### 노드 타입

| 아이콘 | 이름 | 설명 |
|--------|------|------|
| 🏰 | 던전 | Phaser 던전 탐험 화면으로 이동 |
| ⚔️ | 전투 | 일반 적과 전투 (HP: 20) |
| 👹 | 정예 | 강한 적과 전투 (HP: 30) |
| 🔥 | 휴식 | HP 30% 회복 |
| 🛒 | 상점 | 아이템 구매 (미구현) |
| ❓ | 이벤트 | 랜덤 이벤트 발생 |
| 💀 | 보스 | 최종 보스 전투 (HP: 50) |

## 🎯 게임 플레이 가이드

### 1. 맵 화면 (시작)
- 노드를 클릭하여 진행
- 왼쪽 하단: 자원 표시 (금, 정보, 전리품 등)
- 왼쪽 사이드: 에테르 게이지
- 우측 상단: 위험도 표시

### 2. 던전 화면
**조작법:**
- `W/A/S/D`: 이동
- `E`: 상호작용 (문, 상자, 오브젝트)
- `C`: 캐릭터 창 열기/닫기

**목표:**
- 통로와 방을 탐험
- 오브젝트 조사하여 보상 획득
- 출구에 도달하여 맵으로 복귀

### 3. 전투 화면
**조작법:**
- `공격 버튼`: 5-15 랜덤 데미지
- `방어 버튼`: 피해 감소

**목표:**
- 적의 HP를 0으로 만들기
- 플레이어 HP를 0 이상 유지

**보상:**
- 일반 전투: 금 +20, 전리품 +1
- 정예 전투: 금 +20, 전리품 +1
- 보스 전투: 금 +50, 전리품 +1

## 🔧 기술 스택

- **React 18**: UI 컴포넌트 및 상태 관리
- **Phaser 3**: 던전 게임 엔진
- **Babel Standalone**: JSX 변환
- **Tailwind CSS**: 스타일링
- **순수 JavaScript**: 맵 시스템 로직

## 📋 통합 규칙 준수 사항

이 프로젝트는 `통합규칙.md`에 명시된 규칙을 엄격히 준수합니다:

✅ **1. 인터페이스 형태 유지**
- 각 시스템의 원본 UI/UX 완전히 보존
- 던전의 WASD 조작, 전투의 버튼 시스템 등 그대로 유지

✅ **2. 시스템 통합**
- 맵 → 던전 → 맵 순환 구조
- 맵 → 전투 → 맵 순환 구조
- 전역 상태를 통한 데이터 공유

✅ **3. 그래픽 통일**
- 어두운 배경 색상 통일 (#050608, #070409, #111)
- 파란색/보라색 계열 악센트 색상 통일
- 일관된 타이포그래피 (Noto Sans KR)

✅ **4. 내부 디자인 보존**
- 각 시스템의 로직 수정 최소화
- 필요한 연결 부분만 추가

✅ **5. 작동 검증**
- 모든 화면 전환 정상 작동
- 자원 공유 및 지속성 보장
- HP 유지 및 보상 시스템 작동
